Bugs

Emacs21:

Problem: The cursor and mouse pointer are not reset properly when changing to a theme other than the default.

Workaround: Create a new frame after changing themes.

WinNT

Problem: The test script is likely to fail.

Workaround: Install Cygwin and see that Bash and Emacs are properly configured using the cywin32-mount.el package.
